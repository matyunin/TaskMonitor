TaskMonitor
===========
Enterprise solution for dynamic priority management of tasks.

Requirements
------------------------
Debian-based linux
Ruby on rails
MongoDB
PostgreSQL
Nginx

Required gems
------------------------
[haml-rails](https://github.com/indirect/haml-rails)
[devise](https://github.com/plataformatec/devise)
